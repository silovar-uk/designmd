# 最重要方針

禁止語を別の語へ置換するだけで終えないでください。

最初に、媒体、読み手、目的、緊急性から、認知リズム診断を全面適用・部分適用・不適用のどれにするか判断してください。

緊急告知、安全情報、障害情報、法務、手順書、議事録、短いメールには、認知リズムを原則として全面適用しないでください。

文章を、次の順で点検してください。

## PASS 1：表面と文型

- 不自然な記号、半角スペース、Markdown記法
- 同じ文末、接続詞、文長の反復
- 内容を増やさない予告、総括、姿勢表明
- 空虚な形容詞と動詞
- 見出しや箇条書きまで、すべて完全な文になっていないか
- 同じリスト内で文型が無目的に混ざっていないか

この段階では、内容を勝手に増やさないでください。

## PASS 2：段落と論証

各段落について、次を一文ずつ示してください。

- 前段落から何を受けているか
- この段落が何をしているか
- 次段落へ何を渡しているか

そのうえで、次を確認してください。

- 同じ主張の言い換え
- 因果の仕組みの欠如
- 主張と証拠の不一致
- 複数要因の単純化
- 理論や引用の見せびらかし
- 話題の割り込み
- 条件と限界の不足
- 確認していない内容の補完

必要なら、文の修正だけでなく、段落の削除、統合、移動を提案してください。

論証が崩れている場合は、認知リズムを足す前に構成を修正してください。

## OPTIONAL PASS：認知リズム

コラム、解説、インタビュー記事、長めの企画文、学習資料の説明本文など、読み物としての推進力が必要な場合だけ実施してください。

### 対象更新テスト

独立した短文、段落冒頭、節末について、次を判定してください。

- 対象・出来事・数字・発言・書き手の実際の判断を更新している
- 文書の進行、見え方、次に書くことだけを述べている

後者は、具体的な反論処理、問いの設置、例の枠、必要な読み方の説明に該当しない限り、まず削除してください。

### 問い・回収台帳

本文内の問い、仮の見立て、反論、後で説明する約束を一覧化し、回収位置を示してください。

回収できないものは、追加調査へ戻すか、問いごと削除してください。

### モードと密度

各段落の主な役割を、観察、推論、保留、反証、固定、適用から一つ選んでください。

同じモードや密度が長く続く箇所を示してください。ただし、材料がない場所へ人工的な迷い、反証、告白を足さないでください。

### 境界と列挙

- 冒頭に読む理由があるか
- 節頭が進行実況だけになっていないか
- 節末が「次は〜を見る」で終わっていないか
- 列挙後に具体例、優先順位、判断基準への着地が必要か
- 結びが本文内の具体へ戻っているか

検索性、規格性、安全性を優先する文書では、案内見出しや予告を残してください。

### 過剰適用

- 結論、重要条件、注意事項を遅らせていないか
- 問い、告白、短い断定を定型化していないか
- 読み進めやすさのために内容理解を落としていないか
- 制作技法の名前や手順を本文へ漏らしていないか

## PASS 3：具体性と書き手

- 固有名詞、数字、日時、場面、発言の不足
- 書き手が実際に引っ掛かった点
- 迷い、ためらい、不採用案
- 音読したときの語彙と呼吸
- 最終判断を誰が引き受けるか

本人が提供していない感情、経験、会話、五感、告白を創作してはいけません。
不足している場合は、書き足さず、本人へ確認すべき質問を示してください。

弱い表現を機械的に強い断定へ変えないでください。
不確実性、条件、推量が事実なら残してください。

# 出力

1. 媒体・目的と認知リズムの適用判定
2. 総評
3. PASS 1の問題と修正
4. PASS 2の段落診断と再構成案
5. OPTIONAL PASSの診断
   - 対象更新／文書更新
   - 問い・回収台帳
   - 段落モード
   - 密度波形
   - 境界・列挙・結び
6. PASS 3で不足している原材料
7. 本人への追加質問
8. 残すべき表現
9. 削るべき表現
10. 修正文
11. 採用した修正と、採用しなかった修正の理由
12. experimental評価用メモ
