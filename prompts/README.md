# コピペ用プロンプト

必要な部分だけコピーして使えます。

- [`slide-create.md`](slide-create.md)：スライドを新規作成する
- [`slide-redesign.md`](slide-redesign.md)：既存スライドを再設計する
- [`slide-review.md`](slide-review.md)：既存スライドを批評・改善する
- [`writing-review.md`](writing-review.md)：文章のAIくささ、論証、認知リズムを点検する
- [`cognitive-rhythm-writing.md`](cognitive-rhythm-writing.md)：長文のコラム・解説を認知リズムの観点で生成・改稿する（experimental）
- [`visual-review.md`](visual-review.md)：画像・図解の必要性を点検する
- [`final-review.md`](final-review.md)：成果物全体を最終点検する
- [`playful-editoriality.md`](playful-editoriality.md)：通常の制作プロンプトへ追加してあたたかみ・ユーモアを加える（単独では使わず、他のプロンプトへ追加して使う）
- [`question-block-template.md`](question-block-template.md)：逆質問テキストブロックの型だけを単独で使いたい場合

AIがURLを読める場合は、各プロンプトと一緒に次を渡してください。

```text
共通原則：
https://raw.githubusercontent.com/silovar-uk/designmd/main/DESIGN.md
```

認知リズム文章ガイドは、緊急告知、法務、手順書、議事録、短いメールには原則として使いません。プロンプト内の適用判定を先に行ってください。
