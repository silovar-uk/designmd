# 画像・図解レビュー用プロンプト

```text
以下の画像・図解候補を、見栄えではなく情報上の役割で評価してください。

https://raw.githubusercontent.com/silovar-uk/designmd/main/docs/visuals.md

各候補について確認：
- 何の情報を伝えるか
- 削除した場合、何が失われるか
- 証拠、説明、比較、雰囲気のどれか
- 内容固有か、別テーマへ使い回せるか
- 生成AI画像の典型表現へ寄っていないか
- 実写、原文引用、実データ、固有図解の方が適切ではないか
- 誤認の可能性がないか
- 出所、日時、対象、単位、母数が必要か

出力：
- 採用
- 修正して採用
- 不採用
のいずれかと理由。

候補：
（ここに記載）
```
