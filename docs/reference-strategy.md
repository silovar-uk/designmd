# 参照戦略：独自性・面白味・ダイナミックさをつくる

## 参照は「雰囲気」ではなく「操作」を借りる

参考資料を「この感じで」と渡すだけでは、表面的な色、書体、質感、構図が模倣されやすい。

参照資料から抜き出すのは外見ではなく、次のような操作である。

- 何を固定しているか
- 何を変化させているか
- 読む順序をどう作っているか
- どこで視線を止めるか
- どこで規則を破るか
- どの素材同士を衝突させるか
- 情報をいつ明かすか
- 何を繰り返し、何を一度だけ使うか

> 参照資料は完成イメージではなく、判断方法の辞書として使う。

---

## 参照先を四つの役割へ分ける

### 1. 骨格の参照

秩序、編集、グリッド、文字階層を学ぶ。

主な対象：

- 書籍
- 雑誌
- 新聞
- 図録
- 研究誌
- 年次報告書
- サイン計画

借りるもの：

- グリッド
- ベースライン
- 本文、見出し、注、キャプションの役割
- 見開きや章の構造
- 出典と本文の関係
- 余白の機能

借りないもの：

- 紙テクスチャ
- 小さすぎる文字
- 明朝体という表層だけ
- 雑誌風の飾り罫

### 2. 逸脱の参照

秩序を保ったまま、印象を変える方法を学ぶ。

主な対象：

- 実験的タイポグラフィ
- ポスター
- コラージュ
- アートディレクション
- 音楽パッケージ
- 展覧会グラフィック

借りるもの：

- 拡大
- 切断
- 重なり
- 反転
- 斜めの力
- 反復
- 異種素材の衝突
- 一部分だけの規則破り

借りないもの：

- 有名作家の配色や質感の丸写し
- 理由のないランダム配置
- 読みにくさそのもの
- すべてのページを強くすること

### 3. 時間の参照

順序、速度、間、回収を学ぶ。

主な対象：

- 漫画
- 映画タイトル
- 映像編集
- 舞台演出
- 展示動線
- 音楽の構成

借りるもの：

- 静と動の対比
- 情報を明かす順序
- 同じ構図の反復
- 前の要素を次で回収する方法
- 転換点
- 余韻
- 全面、分割、細かな連続の使い分け

借りないもの：

- アニメーションの多用
- 映画的な暗さや照明だけ
- 常に劇的な演出

### 4. 題材固有の参照

その制作物にしかない独自性を得る。

主な対象：

- 元記事
- 文字起こし
- 実際の会話
- 現場写真
- 手書きメモ
- 表、帳票、地図
- 商品や場所の形
- 当事者が使う言葉
- 失敗や違和感

借りるもの：

- 固有の語彙
- 形
- 比率
- 時間
- 記号
- 反復
- その場の温度

ここが最も重要な参照先である。有名デザインは操作方法を与えるが、題材固有の資料が作品の固有性を与える。

制作前に次の表を作る。

|---|---|---|---|---|---|
| 出版物A | 骨格 | 6列グリッド、注釈位置 | 紙色、明朝体 | 研究と会話を並存できる | 文字が小さくなる |

| 漫画C | 時間 | 同じ構図の反復 | コマ枠の模倣 | 解釈の変化を見せられる | 幼く見える可能性 |

# 独自性を作る四層

## 1. 秩序

- グリッド
- 書体体系
- 余白
- 出典
- ページ番号
- 色の役割

秩序は、以後の変化を認識できるようにする背景である。

## 2. 逸脱

資料全体で、一つの逸脱ルールを決める。

例：

- 予想が外れる場所だけ要素がグリッドから外れる
- 反証ページだけ左右を反転する
- 本音の発言だけ欄外へ出す
- 結論が更新されるたびに同じ図が変形する

毎ページ違う奇抜さを入れない。

## 3. 時間

ページ単体ではなく、連続で設計する。

- 静かなページ
- 標準ページ
- 高密度ページ
- 動きのあるページ

を組み合わせる。

資料全体のリズム例：

- 静 → 標準 → 密 → 静 → 動 → 余韻
- 問い → 具体例 → 理論 → 反証 → 再解釈
- 反復 → 小さな変化 → 反復 → 大きな反転

## 4. 固有素材

最低でも2〜3ページに一度、次のいずれかを置く。

- 実際の発言
- 数値
- 日時
- 場所
- 固有名詞
- 現場写真
- 手書き
- 原資料
- 失敗
- 例外

独自性を、珍しい装飾だけに依存させない。

# 面白味を作る方法

面白味は、派手さではなく、予想と発見の差から生まれる。

## 一度目と二度目で意味を変える

- 同じ引用を後半で再掲する
- 最初の図を別の解釈で読み直す
- キャプションが本文へ小さく反論する
- ページ番号や欄外へ小さな物語を持たせる

## 予想を一度だけ外す

1. 読み方の規則を先に覚えさせる
2. 重要な場所で一つだけ外す
3. 外れた理由を内容から理解できるようにする

常に外し続けると、外れは規則になる。

## 異種素材を衝突させる

例：

- 学術論文と会話
- データと手書きメモ
- 公式文書と現場の声
- 完成写真と失敗記録
- 理論図と具体的な一言

衝突には、論点上の理由が必要である。

---

# ダイナミックさを作る操作

ダイナミックさは、装飾量ではなく視線と時間の変化で作る。

使用する操作：

- **拡大**：一語、一数値、一部分を極端に大きくする
- **切断**：画面外へ続く感覚を作る
- **方向**：視線を一方向へ動かす
- **反復**：同じ要素を時間差で変化させる
- **圧縮**：密度の高い領域と空白を隣接させる
- **重なり**：二つの情報の衝突や関係を見せる
- **反転**：前ページの左右、上下、強弱を入れ替える
- **回収**：前に置いた要素の意味を後から変える

原則として、一ページでは主となる操作を一つに絞る。

---

# 参照前の逆質問

1. 絶対に読みやすく保つべき情報は何か。
2. 固定したい要素は何か。
3. 違和感や読みづらさを許せる場所はどこか。
4. ありきたりだと感じる表現は何か。
5. この題材にしかない原材料は何か。
6. 読者の予想を外すなら、どの論点か。
7. 資料全体の速度は、どのように変化させたいか。
8. 参照資料から借りたい操作と、借りたくない外見は何か。

---

# 最終確認

- 参照先の外見ではなく、操作を説明できるか
- 骨格、逸脱、時間、固有素材の四役があるか
- 有名デザインより、題材固有の資料が中心になっているか
- 逸脱ルールは一つの論理でつながっているか
- 静、標準、密、動の差があるか
- すべてのページを面白くしようとしていないか
- 一度目より二度目に意味が増える要素があるか
- 別テーマへそのまま使い回せないか
