# 画像・図解ガイドライン

対象：写真、イラスト、生成画像、アイコン、図解、グラフ、チャート、KV、サムネイル。

## 画像の仕事

画像には、事実、対象特定、構造、比較、時間、空間、記憶、意図した感情設計のいずれかを担わせる。

> この画像を削除したら、失われる情報は何か。

答えが「見栄え」だけなら削除候補。

## 生成画像の典型に注意する

- 暗い背景
- 発光する人物
- 粒子
- ニューラルネットワーク
- ワイヤーフレーム
- 光の筋
- 人物間のエネルギー
- 抽象的な脳
- 顔のないシルエット
- 過度な映画照明
- 赤、青、金の高コントラスト
- 象徴的すぎる仮面、道化師、影

禁止ではない。内容固有の理由なしに使わない。

## 実写も自動的に人間らしくはならない

撮影場所、対象者、日時、行動が主張と関係するか確認する。証拠なのか演出なのかを区別する。

## 図解

悪い図解：本文の3項目を3つの丸へ入れる、抽象語を矢印でつなぐ、アイコンを置くだけ、色分けに意味がない。

良い図解：前後、原因と結果、選択肢の差、優先順位、情報の流れ、例外を新たに見せる。

## チャート

作成前に、比較対象、判断事項、期間、単位、母数、基準値、例外を決める。

3D、不要なグラデーション、装飾アイコン、読めない凡例、切断軸、過剰な色、数値のない印象図を避ける。

## 生成画像の適切な用途

実在しないコンセプト、将来像、権利上実写を使えない場合、ストーリーボード、初期の方向性検討、世界観探索。

最終利用前に、内容固有性、既視感、文字・手・構造の破綻、事実画像との誤認、他素材より適切な理由を確認する。必要に応じて「イメージ」「生成画像」と明示する。

## 素材の優先順位

1. 実際の現場写真・画面
2. 原文・発言・資料の抜粋
3. 実データの図表
4. 固有の説明図
5. ライセンス確認済み写真・イラスト
6. 生成画像
7. 汎用アイコン・装飾
