# 品質ルーブリック

[`review-checklist.md`](review-checklist.md) をはじめとする既存のチェックリスト群は、0（問題なし）・1（少し該当）・2（明確に該当）で「AIくささ」という悪い兆候を検出する形式である。これに対し本ファイルは、完成した成果物がどのレベルの品質に到達しているかを5段階（1・3・5点をアンカーとする）で判定する、到達水準のルーブリックである。チェックリストが穴を探す道具であるのに対し、本ファイルは水準を測る道具であり、役割が異なる。点数を上げること自体を目的にしない。点数は、どの軸が弱く、どのSTEPへ戻って作り直すべきかを特定するために使う。

対象は2種類ある。前半は成果物（実際に制作したスライド・文書・学習教材など）を評価する10軸、後半はガイド自体（design.md群）の完成度を自己評価する5軸である。両者は評価対象が異なるため、混同しない。

---

## 第1部：成果物評価（10軸×5段階）

各軸を1〜5点で採点する。2点・4点は1点と3点、または3点と5点の中間状態として扱う。

### ①中心メッセージの明確さ

- 1点：資料全体を貫く一文が存在しない、または複数の主張が混在している
- 3点：中心メッセージは存在するが、一部のページがそれと無関係な情報を含んでいる
- 5点：30字以内の一文で言い切れる中心メッセージがあり、全ページがそれを支持するか、反例テストに耐えている

### ②ストーリーの必然性

- 1点：「表紙→アジェンダ→本論→まとめ」のような既定の型をそのまま採用しており、他の構成案と比較した形跡がない
- 3点：構成案を複数比較した記録はあるが、採用理由が「見た目が整っている」など内容と無関係な基準にとどまる
- 5点：採用した構成が、目的・読み手・原材料のいずれか一つでも変われば成立しない、この題材固有の必然性を持つ理由を説明できる

### ③1ページ1主張の徹底度

- 1点：1ページに独立した主張が複数並び、どれが本題か判別できないページが複数ある
- 3点：大半のページは1主張だが、一部のページに主張と無関係な補足情報が混在している
- 5点：全ページが単一の主張を持ち、見出しだけを追っても論理が途切れず進む

### ④情報の優先順位（削る判断）

- 1点：削除基準がなく、集めた原材料をほぼそのまま掲載している
- 3点：削除基準はあるが実際に削られた情報が少なく、密度の高いページが目立つ
- 5点：中心メッセージ・読後の変化・題材固有性のいずれにも該当しない情報が本編から削られ、どこ（付録・台本・制作ログ）へ移動したか説明できる

### ⑤冒頭の掴み

- 1点：冒頭が表紙とアジェンダの既定パターンで始まり、なぜその入り方を選んだか説明できない
- 3点：冒頭の型は意図的に選ばれているが、具体的な事実・数字・発言ではなく抽象的な問いや一般論から始まっている
- 5点：冒頭の型が読み手の出発点（納得済み／課題未自覚／時間がない）から逆算されており、具体的な事実・数字・発言から始まり、意味が早い段階で回収される

### ⑥転換と見せ場の設計

- 1点：転換点や見せ場が存在しない、または資料全体が同じ強度で書かれ、どこが山場か判別できない
- 3点：転換・見せ場は配置されているが、視覚上の変化（密度・配色・レイアウト）と意味の変化が同じ場所で起きていない、または見せ場が連続している
- 5点：転換点（1〜3箇所）で視覚と意味の変化が同期しており、見せ場の直前に低密度ページが置かれ、8〜12枚に1回程度の間隔が保たれている

### ⑦密度リズム（静・動の差）

- 1点：全ページがほぼ同じ密度で書かれており、同じ密度のページが4枚以上連続する箇所がある
- 3点：密度に差はあるが密度リズム譜として意図的に設計された形跡がなく、高密度ページの連続が残っている
- 5点：静・標準・密・動の状態が資料内に配置され、同じ密度が4枚以上連続する箇所がなく、密度の落差が転換・見せ場の到達感を支えている

### ⑧タイポグラフィ（ジャンプ率・行長など数値面）

- 1点：サイズ段階が5段階を超える、またはジャンプ率が全ページで同一のまま資料の温度感と無関係に使われている
- 3点：サイズ段階・行長はおおむね目安内だが、転換点や見せ場でジャンプ率を意図的に変える調整がされていない
- 5点：サイズ段階が3〜4段階に絞られ、行長・行間が資料形式の目安値に収まり、転換点・見せ場でジャンプ率が意図的に変化している

### ⑨レイアウトの意図（グリッドと逸脱の必然性）

- 1点：グリッドが一貫しておらず、ページごとに要素配置の基準が変わる、またはグリッド逸脱の理由が説明できない
- 3点：基本グリッドは一定だが、逸脱が転換・見せ場・引用の3箇所以外にも現れている、または同一レイアウトが4枚以上連続している
- 5点：基本グリッドが1つに定まり、逸脱は転換・見せ場・引用の3箇所に限定され、逸脱ごとの内容上の理由が制作ログに残っている

### ⑩写真・図版・図表の必然性（削除テストへの耐性）

- 1点：画像を削除しても本文の理解が変わらない、または生成画像特有の意匠（発光、粒子、ワイヤーフレーム等）が内容固有の理由なく使われている
- 3点：一部の画像は削除テストに耐えるが、装飾目的で置かれた図版が残る、またはチャートの単位・母数・出典が不明な箇所がある
- 5点：全ての画像・図表が削除テストに耐え（削除すると理解が損なわれる）、チャートは1メッセージ1チャートで結論型タイトルを持ち、出典・単位・母数が明記されている

## 第2部：ガイド自己評価（5軸）

design.md群自体の完成度を評価する軸である。成果物の評価とは対象が異なる。各軸1〜5点で採点する。

### ⑪実行可能性

- 1点：記述が方針や理念の表明にとどまり、AIがそこから制作判断を下せない
- 3点：一部の項目に数値や判断基準があるが、多くは抽象語のままで読み手が適用方法を補う必要がある
- 5点：各項目に具体的な数値・条件・手順があり、この記述だけを根拠に採用・不採用を判断できる

### ⑫誤読しにくさ

- 1点：一つの文が複数の解釈を許し、適用条件や例外が書かれていないため読み手によって結論が変わる
- 3点：大半の記述は一意に読めるが、一部の語（「適切に」「必要に応じて」等）が条件を伴わず使われている
- 5点：全ての規則に適用条件・例外・使用場面が明記されており、別の読み手が読んでも同じ制作判断に至る

### ⑬ファイル間の整合性

- 1点：複数ファイルが同じ主題について矛盾する基準を示している、または同じ内容が言い換えを変えて重複している
- 3点：明確な矛盾はないが、どちらのファイルが優先されるか委譲関係が示されていない重複がある
- 5点：各主題の正典（委譲元）が1ファイルに定まっており、他ファイルはそこへの参照のみで矛盾・重複がない

### ⑭継続的な更新のしやすさ

- 1点：一つの規則を変更すると、他の複数ファイルの記述も連動して書き換える必要がある
- 3点：主要な規則は独立して更新できるが、一部の数値や用語が複数ファイルに直接書き込まれている
- 5点：各規則が1箇所にのみ定義され、関連ファイルは参照のみで、変更が1ファイルの編集で完結する

### ⑮異なる資料タイプへの汎用性

- 1点：記述がスライドなど特定の資料形式を前提にしており、文書や学習教材など別形式に適用すると規則が破綻する
- 3点：大半の規則は形式を問わず適用できるが、一部の数値や手法が特定形式専用のまま、他形式向けの読み替えが示されていない
- 5点：各規則が資料形式ごとの数値・使い分けを併記しており、スライド・文書・学習教材のいずれにも同じ手順で適用できる

## 運用規則

### 出荷不可の判定

成果物評価10軸（①〜⑩）のうち、3点以下の軸が1つでもあれば「出荷不可」とする。表面的な修正で点数を底上げせず、該当軸に対応するSTEPへ戻って作り直す。

### 戻るSTEP対応表

| 軸 | 戻るSTEP |
|---|---|
| ①中心メッセージの明確さ | STEP 3（中心メッセージ） |
| ②ストーリーの必然性 | STEP 4（ストーリーライン） |
| ③1ページ1主張の徹底度 | STEP 5（各ページの役割と主張） |
| ④情報の優先順位 | STEP 6（情報量と優先順位） |
| ⑤冒頭の掴み | STEP 4（ストーリーライン） |
| ⑥転換と見せ場の設計 | STEP 5（各ページの役割と主張） |
| ⑦密度リズム | STEP 10（リズムと完成度検証） |
| ⑧タイポグラフィ | STEP 8（レイアウトと文字組み） |
| ⑨レイアウトの意図 | STEP 8（レイアウトと文字組み） |
| ⑩写真・図版・図表の必然性 | STEP 6（情報量と優先順位） |

### 平均点による差し戻し

成果物評価10軸の平均点が4.0未満の場合、出荷不可の軸が無くても、STEP 10（リズムと完成度検証）からやり直す。個々の軸が最低ラインを超えていても、全体としての完成度が水準に届いていないことを意味するためである。
