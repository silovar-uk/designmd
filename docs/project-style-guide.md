# 案件別スタイルガイド

> 状態：stable  
> designmd版：2026.07.19-beta  
> 最終更新：2026-07-19

`designmd` は、文章・スライド・画像制作に共通する判断基盤である。個別案件のブランド、配色、書体、余白、語り口を固定するテンプレートではない。

> 共通基盤は、判断の下限をそろえる。案件別スタイルガイドは、今回の問いと表現を選ぶ。

この区別は、デジタル庁デザインシステムが、プラットフォーム型の仕組みと個別サービス向けのスタイルガイドを分けている考え方を、designmd向けに再構成したものである。

## 1. 上書きできない共通基盤

案件別スタイルガイドでも、次は上書きしない。

- 与えられた問いを監査し、採用理由を残す
- 事実・解釈・仮説・提案を区別する
- 出典、条件、限界を主張の近くに置く
- 読み手が理解・判断・行動できる最低限のアクセシビリティを守る
- 読者向け本編、台本、付録、制作ログを分ける
- AIやガイドへ最終判断を移譲しない
- 旧版を再設計するとき、理解上の機能を失わせない
- 議事録では、内容説明型の日本語見出しだけを使い、あおり文、英語見出し、英語副題を使わない

## 2. 案件ごとに決めるもの

- 読み手と使用場面
- 資料モード：説明・報告／説得・提案／読み物・体験／混合
- 読後の変化
- 採用した問い
- 採用しなかった問いと理由
- 中心メッセージ
- 固有素材
- 語り口と文体
- スライドタイトルの基本強度と、強いタイトルを使うページ
- 書体、文字階層、色、余白、グリッド
- 画像・図・表の扱い
- 読み物では、主となる編集型、副型、読み方、密度のリズム
- 素材設計では、想定する物体、五つの質感軸、制作痕跡
- 逸脱ルール
- 媒体別のアクセシビリティ対応
- 固定するもの、変えてよいもの

資料モードは、見た目の雰囲気ではなく、読み手に何を理解、判断、実行してほしいかで決める。提案資料であっても、事実ページまで強い断言へ変換しない。

読み物・体験または混合モードでは、「雑誌風」を外見の指定として扱わない。読後の変化、強い原材料、求める読み方から、[`editorial-design.md`](editorial-design.md)の主型と副型を選ぶ。

質感は、紙の粒子を足す指定として扱わない。[`materiality.md`](materiality.md)に従い、表面、反射、精度、時間、重量と、最大二つの制作痕跡へ分解する。

議事録は例外なく説明・報告の文書として扱う。見出しは「券売」「広告」「決定事項」「対応事項」など、内容を直接示す日本語にする。あおり文、キャッチコピー、比喩、英語見出し、英語副題は案件側の指定でも採用しない。

```text

採用理由：
採用しなかった問い：
中心メッセージ：
固有素材：
語り口：
タイトル設計：内容説明型／解釈併記型／提案・判断型の使い分け
強いタイトルを使うページ：
視覚の骨格：
編集型の主型：
編集型の副型と使用箇所：
読ませ方：拾い読み／連続閲覧／熟読／探索／段階理解
最初の入口：
密度のリズム：
想定する物体：
質感五軸：表面 / 反射 / 精度 / 時間 / 重量
制作痕跡：最大二つ
画面への翻訳：
一つの逸脱ルール：
逸脱が示す内容上の転換：
アクセシビリティ最低条件：
固定するもの：

```

編集型と素材設計は、読み物・体験または混合モードで必要な場合だけ使う。短い業務文書へ無理に追加しない。

## 4. 詳細版で追加する項目

| 項目 | 決める内容 | 決めない場合のリスク |
|---|---|---|
| 問い | 前提、根拠、三つの代替問い、採否 | 表現だけが先に決まる |
| 資料モード | 説明・報告、説得・提案、読み物・体験、混合 | 全ページが同じ語調になる |
| タイトル | ページ役割、タイトル強度、強いタイトルを使う場所 | 事実と提案の声量が混ざる |
| 議事録 | 日本語の内容説明型見出し、決定・方針・検討・確認の区分 | あおり文や英語が記録の機能を邪魔する |
| 編集型 | 主型、副型、読み方、最初の入口、密度のリズム | 雑誌らしさが書体や段組みだけになる |
| 素材設計 | 想定物体、素地、インク、光、構造、経年、五軸 | 紙テクスチャや汚しが意味なく増える |
| 制作痕跡 | 印刷、コピー、切り貼り、折り等から最大二つ | ページごとに別の加工が増える |
| タイポグラフィ | 書体、サイズ段階、行長、行間、強調方法 | ページごとに声量が変わる |
| 色 | ベース、アクセント、意味色、白黒時の代替 | 色だけに意味を依存する |
| 余白 | 外周、関連項目間、セクション間 | 関係が読み取れない |
| 写真 | 撮り下ろし、支給、生成、トリミング規則 | 素材の出所と役割が混ざる |
| 図・表 | 何を図示し、何を表で残すか | 正確さと印象が混同される |
| 台本 | 文体、長さ、視覚情報の説明方法 | スライド本文の読み上げになる |
| アクセシビリティ | 読み順、代替説明、コントラスト、字幕等 | 最終工程で修正不能になる |

## 5. 作るタイミング

- 定める：問いの監査、読み手、使用場面、資料モード、固定条件を仮置きする
- 集める：問いを支える証拠と反証を加える。写真、紙、印刷などの素材候補も出所と役割を記録する
- 組み立てる：中心メッセージ、物語、ページ役割、タイトル強度を反映する。読み物では編集型の候補を三案比較する
- つくる：書体、色、余白、グリッド、主型・副型、質感五軸、制作痕跡、逸脱ルールを確定する
- 確かめる：代表ページとタイトル一覧で、問い、資料モード、編集型、素材設計が結びついているか確認する

案件別スタイルガイドは最初に完成させる文書ではない。制作判断に合わせて更新し、代表ページが成立した時点で固定する。

## 6. 使わない方がよいケース

- 1ページの簡単なメモで、既存書式をそのまま使う
- 誤字修正のみで、表現や構造を変更しない
- 緊急時の短い連絡で、ブランド表現より正確さと速度が優先される
- 議事録、手順書、注意事項に、雑誌型や制作痕跡を持ち込む

ただし、事実確認、責任境界、誤解防止は省略しない。
