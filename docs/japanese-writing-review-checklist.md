# 日本語AI文章レビュー・チェックリスト

目的は、AI判定の回避ではなく、日本語としての自然さ、論証の筋、書き手の判断を取り戻すことです。

## PASS 0：媒体と前提

- [ ] 誰から誰への文章か分からない
- [ ] 記事、報告書、プレゼン、メールなどの媒体差がない
- [ ] 読み手に求める理解、判断、行動が曖昧
- [ ] 必要な証拠と、書き手固有の原材料が整理されていない

## PASS 1：表面とリズム

- [ ] 不自然な半角スペースやMarkdown記法が残る
- [ ] 日本語文脈で不自然なコロン、ダッシュ、区切り線がある
- [ ] 同じ文末や接続詞が続く
- [ ] 類義語による無意味な強調がある
- [ ] 導入や締めが定型文になっている
- [ ] 内容を増やさない予告、総括、姿勢表明がある
- [ ] 作用を曖昧にする「実現する」「提供する」などが続く
- [ ] 見出し、ラベル、箇条書きまですべて完全な文
- [ ] 同じリスト内で名詞句、事実文、行動文が無目的に混ざる

## PASS 2：段落と論証

- [ ] 一段落に複数のトピックが混ざる
- [ ] 段落の役割を一文で説明できない
- [ ] 前段落から何を受け、次へ何を渡すか分からない
- [ ] 予告、説明、総括で同じ主張を繰り返す
- [ ] 原因と結果の間の仕組みがない
- [ ] 複数要因を一つの原因へ丸めている
- [ ] 主張と証拠が別の問いに答えている
- [ ] 理論や引用を置いたが、その後の判断が変わらない
- [ ] 話題が途中で割り込み、論証が一方向に進まない
- [ ] 確認していない内容を滑らかに補っている
- [ ] 条件と限界が主張から離れている

## PASS 3：具体性と書き手

- [ ] 評価語だけで、何がどう違うか書いていない
- [ ] 誰が、いつ、どこで、何をしたか分からない
- [ ] 数字、期間、比較条件、実際の発言がない
- [ ] 書き手がどこに引っ掛かったか分からない
- [ ] 音読すると、本人が使わない語彙や呼吸がある
- [ ] 比喩が具体的な経験と結びつかない
- [ ] 迷い、ためらい、不採用案が消えている
- [ ] 本人が提供していない感情、経験、場面を加えている
- [ ] 最終的に誰が判断を引き受けるか分からない

## 危険信号

次に該当する場合、語尾修正ではなく前工程へ戻ります。

- 原材料が一般論しかない
- 主張を支える証拠がない
- 段落の役割を説明できない
- 読み手と媒体が未定
- 禁止語の類義語置換だけを行っている
- 本人が「自分の言葉ではない」と感じる

## 戻る工程

- 媒体、読み手、目的が曖昧：文体契約へ戻る(STEP 1)
- 具体性がない：原材料収集と逆質問へ戻る(STEP 2)
- 論証が崩れる：構成と段落設計へ戻る(STEP 4)
- 文末や記号だけが不自然：表面校正へ戻る(STEP 9)
- 書き手の気配がない：本人による音読と書き直しへ戻る(STEP 9)

総合的な品質水準の判定は `docs/quality-rubric.md` の⑬日本語の自然さ相当の観点(実際のルーブリックでは①〜⑩の成果物評価軸を参照)を併用する。

詳細は [`japanese-ai-writing-practices.md`](japanese-ai-writing-practices.md) を参照してください。
