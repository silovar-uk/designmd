# 日本語AI文章レビュー・チェックリスト

目的は、AI判定の回避ではなく、日本語としての自然さ、論証の筋、書き手の判断を取り戻すことです。

## PASS 0：媒体と前提

- [ ] 誰から誰への文章か分からない
- [ ] 記事、報告書、プレゼン、メールなどの媒体差がない
- [ ] 読み手に求める理解、判断、行動が曖昧
- [ ] 必要な証拠と、書き手固有の原材料が整理されていない
- [ ] 認知リズムを使うべき文書か、適用しない文書かを判定していない

## PASS 1：表面とリズム

- [ ] 不自然な半角スペースやMarkdown記法が残る
- [ ] 日本語文脈で不自然なコロン、ダッシュ、区切り線がある
- [ ] 同じ文末や接続詞が続く
- [ ] 類義語による無意味な強調がある
- [ ] 導入や締めが定型文になっている
- [ ] 内容を増やさない予告、総括、姿勢表明がある
- [ ] 作用を曖昧にする「実現する」「提供する」などが続く
- [ ] 見出し、ラベル、箇条書きまですべて完全な文
- [ ] 同じリスト内で名詞句、事実文、行動文が無目的に混ざる

## PASS 2：段落と論証

- [ ] 一段落に複数のトピックが混ざる
- [ ] 段落の役割を一文で説明できない
- [ ] 前段落から何を受け、次へ何を渡すか分からない
- [ ] 予告、説明、総括で同じ主張を繰り返す
- [ ] 原因と結果の間の仕組みがない
- [ ] 複数要因を一つの原因へ丸めている
- [ ] 主張と証拠が別の問いに答えている
- [ ] 理論や引用を置いたが、その後の判断が変わらない
- [ ] 話題が途中で割り込み、論証が一方向に進まない
- [ ] 確認していない内容を滑らかに補っている
- [ ] 条件と限界が主張から離れている

## OPTIONAL PASS：認知リズム

コラム、解説、インタビュー記事、長めの企画文、学習資料の説明本文など、読み物としての推進力が必要な場合だけ確認する。

緊急告知、安全情報、法務、手順書、議事録、短いメールには原則として適用しない。

### 対象更新

- [ ] 独立した短文が、対象の情報ではなく文章の進行だけを述べている
- [ ] 段落冒頭が「ここまで見てきた」「次に確認する」だけになっている
- [ ] 節末が「次は〜を見る」という予定の実況で終わっている
- [ ] 内容を増やさない総括を、短い決め台詞へ変えて残している
- [ ] 制作技法の名前や操作を本文へ漏らしている

### 問いと回収

- [ ] 問いを立てたが、その後の説明や判断が変わらない
- [ ] 問い、仮の見立て、後で説明する約束の回収位置を指せない
- [ ] 答えがない問いを、思わせぶりなまま放置している
- [ ] 一つの節に主要な問いを開きすぎている
- [ ] 読者への問いかけを、文章を始める定型として使っている

### モードと密度

- [ ] 観察、推論、保留、反証、固定、適用のうち、一つの役割が長く続く
- [ ] 具体的な数字・発言・場面から離れたまま、抽象説明が続く
- [ ] 密度を下げる文が、対象について何も新しく伝えていない
- [ ] 短文を増やしただけで、読み手の認識が変化していない
- [ ] 材料がない場所へ、人工的な迷い、反証、告白を足している

### 境界と着地

- [ ] 冒頭に読む理由がなく、議題表だけになっている
- [ ] 理論・概念が、具体的な違和感より先に出ている
- [ ] 列挙後に、今回の具体例・優先順位・判断基準へ戻る必要があるのに放置している
- [ ] 逆に、参照用一覧やチェックリストへ不要な着地文を足している
- [ ] 結びが、どのテーマにも使える一般論で終わっている
- [ ] 冒頭の具体や本文中の判断を、更新後の見方で読み直していない

### 過剰適用

- [ ] 結論、重要条件、注意事項が遅くなっている
- [ ] すべての文章が「問い→意外な事実→短い断定」の型になっている
- [ ] 不必要な告白、二人称、反問が増えている
- [ ] 読み進めやすさは上がったが、内容理解が落ちている
- [ ] 修正時間が効果に比べて増えすぎている

## PASS 3：具体性と書き手

- [ ] 評価語だけで、何がどう違うか書いていない
- [ ] 誰が、いつ、どこで、何をしたか分からない
- [ ] 数字、期間、比較条件、実際の発言がない
- [ ] 書き手がどこに引っ掛かったか分からない
- [ ] 音読すると、本人が使わない語彙や呼吸がある
- [ ] 比喩が具体的な経験と結びつかない
- [ ] 迷い、ためらい、不採用案が消えている
- [ ] 本人が提供していない感情、経験、場面を加えている
- [ ] 最終的に誰が判断を引き受けるか分からない

## 危険信号

次に該当する場合、語尾修正ではなく前工程へ戻ります。

- 原材料が一般論しかない
- 主張を支える証拠がない
- 段落の役割を説明できない
- 読み手と媒体が未定
- 禁止語の類義語置換だけを行っている
- 本人が「自分の言葉ではない」と感じる
- 論証を直さず、問いと短文で読ませようとしている
- 書き手が持っていない迷い・感情・告白を作っている
- 緊急性や検索性が必要な文章で、結論を後ろへ送っている

## 戻る工程

- 媒体、読み手、目的が曖昧：文体契約へ戻る（STEP 1）
- 具体性がない：原材料収集と逆質問へ戻る（STEP 2）
- 論証が崩れる：構成と段落設計へ戻る（STEP 4）
- 平坦だが論証は成立している：認知リズム診断を任意適用する（STEP 9）
- 文末や記号だけが不自然：表面校正へ戻る（STEP 9）
- 書き手の気配がない：本人による音読と書き直しへ戻る（STEP 9）

総合的な品質水準の判定は `docs/quality-rubric.md` の成果物評価軸を併用する。
