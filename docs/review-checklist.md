# AIくささレビュー・チェックリスト

各項目を0（問題なし）、1（少し該当）、2（明確に該当）で確認する。点数は検出器ではなく、改善箇所の発見に使う。

## 共通基盤・案件別スタイル

- [ ] designmdの共通基盤を、そのまま案件の外見テンプレートにしている
- [ ] 案件別スタイルガイドがない
- [ ] 読み手、語り口、書体、色、余白、逸脱の理由を説明できない
- [ ] 案件固有の条件を、共通ルールとして固定している
- [ ] 新しい規範に、向く場面と注意が必要な場面がない

## アクセシビリティ

- [ ] アクセシビリティを完成後のコントラスト確認だけにしている
- [ ] 色、位置、アニメーションだけで意味を伝えている
- [ ] 見出し、読み順、オブジェクト順が意味と一致していない
- [ ] 画像、図、表、動画の要点に代替説明がない
- [ ] 実寸、白黒、拡大、読み上げ、キーボード等の必要な確認をしていない
- [ ] designmdを使ったことを、規格適合の保証として扱っている

## 役割・責任

- [ ] 目的、権利、公開可否、最終承認をAIやガイドへ移譲している
- [ ] 誰がファクトチェック、アクセシビリティ、公開可否を確認するか決まっていない
- [ ] 最終判断者が成果物を確認していない

## Experimental

- [ ] experimentalなルールをstableと同じ強さで適用している
- [ ] 形式や数値を先に決め、内容を当てはめている
- [ ] 試行件数、評価方法、撤回条件がない
- [ ] ガイド適用前より、制作時間だけが増えている
- [ ] 理解改善を確認せず、ルールを増やしている

## 段階制作・早すぎる収束

- [ ] 原材料から、いきなり完成版を作っている
- [ ] 目的や読み手を確認する前に、テイストを決めている
- [ ] 構成案を比較せず、最初の案を採用している
- [ ] 構成が未確定のまま、細かな装飾を作り込んでいる
- [ ] グリッドや文字体系を試す前に、全ページを量産している
- [ ] 静、標準、密、動の代表ページを試していない
- [ ] 重要な段階でユーザーまたは制作者の判断を挟んでいない
- [ ] 完成後の自己評価だけで、途中の判断を検証していない

## 文章

- [ ] 抽象語が具体語より多い
- [ ] 「AではなくB」が複数回ある
- [ ] すべての見出しがキャッチコピー風
- [ ] 文の長さが均一
- [ ] 同じ接続詞が続く
- [ ] 結論が毎回きれいにまとまりすぎる
- [ ] 固有名詞、数字、日時が少ない
- [ ] 実際の発言や観察がない
- [ ] 誰にでも当てはまる一般論が多い
- [ ] 書き手の判断や迷いが見えない

## 認知リズム

- [ ] 独立短文、段落冒頭、節末が、対象ではなく文章の進行だけを述べている
- [ ] 問いを置いたが、後の説明や判断が変わらない
- [ ] 問い、仮の見立て、後で説明する約束の回収位置を指せない
- [ ] 観察、推論、保留、反証、固定、適用の一つだけが長く続く
- [ ] 具体へ戻らず、抽象説明が続く
- [ ] 短文を増やしただけで、読み手の認識が変わっていない
- [ ] 列挙が、具体例、優先順位、判断へ着地していない
- [ ] 結びが、本文中の具体へ戻らず一般論で終わっている
- [ ] 結論、重要条件、注意事項を、読ませるために遅らせている
- [ ] 問い、告白、短い断定が新しいテンプレートになっている
- [ ] 書き手が持っていない迷い、感情、告白を作っている
- [ ] 認知モードや密度波形を、内容ではなく固定配列として使っている

詳細は [`cognitive-rhythm-writing.md`](cognitive-rhythm-writing.md) と [`cognitive-rhythm-review.md`](cognitive-rhythm-review.md) を参照する。

## 構造・レイアウト

- [ ] 同じ構成が3ページ以上続く
- [ ] すべて3項目に整理されている
- [ ] 内容に関係なくカードを使う
- [ ] 全ページの情報密度が同じ
- [ ] ページの役割が区別できない
- [ ] 英語ラベルが装飾になっている
- [ ] ウェブサイト風UIをそのまま使う
- [ ] 読み手が何を判断すべきか分からない
- [ ] 発表用、閲覧用、会議用の区別がない
- [ ] グリッドが関係の設計ではなく、同じ箱の反復になっている
- [ ] 前後のページで理解がどう変わるか説明できない

## 形式選択

- [ ] 箇条書きで十分な並列項目をカード化している
- [ ] カード内の項目が独立した入口になっていない
- [ ] 詳細な数値比較をカードで行っている
- [ ] 見出しが原文の流れを不必要に分断している
- [ ] 正確な値を読ませたいのに、印象的なチャートだけにしている
- [ ] 画像を消しても「見栄え」しか失われない
- [ ] 図が本文を丸や矢印へ入れ直しただけになっている

## 参照・独自性

- [ ] 参照資料を「この感じで」という雰囲気だけで使っている
- [ ] 借りる操作と借りない外見を区別していない
- [ ] 骨格、逸脱、時間、固有素材の参照が一つの種類に偏っている
- [ ] 有名作品の配色、書体、質感を表面的に模倣している
- [ ] 題材固有の発言、数字、写真、表、手書きが中心になっていない
- [ ] 別テーマへそのまま使い回せる
- [ ] 一度目より二度目に意味が増える要素がない
- [ ] 面白味を、奇抜な装飾だけで作っている

## リズム・ダイナミックさ

- [ ] すべてのページが同じ速度で読まれる
- [ ] 低密度、標準密度、高密度の差がない
- [ ] すべてのページを強く、派手にしている
- [ ] 逸脱ルールが複数あり、ページごとに別の奇抜さを使う
- [ ] 逸脱と内容上の転換が結びついていない
- [ ] 拡大、切断、反復、反転などを一ページで使いすぎる
- [ ] 静かなページや余韻がない
- [ ] 前に置いた要素の反復や回収がない

## 画像・図

- [ ] 画像を消しても理解が変わらない
- [ ] 抽象概念を抽象画像へ置き換える
- [ ] 発光、粒子、ワイヤーフレームに依存する
- [ ] 図解が本文の再掲になっている
- [ ] アイコンや色分けに意味がない
- [ ] 写真の日時、場所、出所が不明
- [ ] チャートの単位、母数、期間が不明
- [ ] 生成画像を事実画像と誤認しうる
- [ ] 別テーマでも使い回せる素材である
- [ ] 紙テクスチャ、版ずれ、手書き風を人間らしさの代用品にしている

## 根拠・論理

- [ ] 事実と解釈が混ざる
- [ ] 強い主張の根拠が遠い
- [ ] 反証がない
- [ ] 限界が最後に小さくまとめられる
- [ ] 対象範囲、時点、比較条件が不明
- [ ] 因果と相関を混同する
- [ ] 参考文献が内容と結びついていない
- [ ] 出典が特定言語、特定分野へ偏る

## 判断主体

- [ ] 誰が結論を選んだか分からない
- [ ] なぜこの構成、グリッド、画像なのか説明できない
- [ ] 不採用案がない
- [ ] AIの初稿をほぼそのまま使う
- [ ] 読み手固有の情報がない
- [ ] 使用場面や制約が反映されていない
- [ ] 制作者の観察が残っていない
- [ ] 最終判断者が不明
- [ ] 各ステージで何を確定したか記録がない
- [ ] 代表ページで何を確認したか説明できない

## 点数より先に見る危険信号

以下のいずれかに該当する場合、表面的な修正ではなく前のSTEPへ戻る。

- 読み手、使用場面、読後の変化が不明
- 案件別スタイルガイドがない
- 構成案を比較していない
- 代表ページを試していない
- 題材固有の原材料がない
- 参照先から借りた操作を説明できない
- 逸脱の理由を説明できない
- 色や位置だけで重要情報を伝えている
- 論証を直さず、問いと短文で読ませようとしている
- 最終判断者が不明

## 戻るステージの目安

- 目的、読み手、閲覧環境が曖昧：STEP 1へ戻る
- 具体性、根拠、代替説明の材料が弱い：STEP 2へ戻る
- 中心メッセージが曖昧：STEP 3へ戻る
- 話が平坦、論理が弱い：STEP 4〜5へ戻る
- 情報量、認知負荷が高い：STEP 6へ戻る
- 既視感、形式先行、案件固有性不足：STEP 7へ戻る
- 読み順、実寸、アクセシビリティ実装が弱い：STEP 8へ戻る
- 論証は成立しているが文章が平坦：STEP 9の認知リズムへ戻る
- AI的な均質さが残る：STEP 9へ戻る
- 装飾過多、重複、リズム不良：STEP 10へ戻る

## 最終17問

1. この成果物でしか伝えられない情報は何か
2. 制作者が実際に判断した箇所はどこか
3. 読み手に何を判断、理解、実行してほしいか
4. どこまでが事実で、どこからが解釈か
5. なぜこの文章、配置、画像でなければならないか
6. 完成前に、どの段階で何を検証したか
7. designmdの共通基盤と案件固有の選択を分けられるか
8. 案件別スタイルガイドで何を固定し、何を変えたか
9. 参照先から何の操作を借り、何の外見を借りなかったか
10. 文章の進行だけを実況する文が残っていないか
11. 問いが後の説明または判断を変えたか
12. 認知モードと密度を、新しい固定テンプレートにしていないか
13. グリッドは何を関係づけているか
14. 逸脱は、どの内容上の転換を表しているか
15. 色、位置、動きを失っても意味が伝わるか
16. experimentalな判断は、何をもって成功・撤回とするか
17. 正確性、権利、アクセシビリティ、公開可否の最終判断者は誰か
