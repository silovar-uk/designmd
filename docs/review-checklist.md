# AIくささレビュー・チェックリスト

各項目を0（問題なし）、1（少し該当）、2（明確に該当）で確認する。点数は検出器ではなく、改善箇所の発見に使う。

## 文章

- [ ] 抽象語が具体語より多い
- [ ] 「AではなくB」が複数回ある
- [ ] すべての見出しがキャッチコピー風
- [ ] 文の長さが均一
- [ ] 同じ接続詞が続く
- [ ] 結論が毎回きれいにまとまりすぎる
- [ ] 固有名詞、数字、日時が少ない
- [ ] 実際の発言や観察がない
- [ ] 誰にでも当てはまる一般論が多い
- [ ] 書き手の判断や迷いが見えない

## 構造・レイアウト

- [ ] 同じ構成が3ページ以上続く
- [ ] すべて3項目に整理されている
- [ ] 内容に関係なくカードを使う
- [ ] 全ページの情報密度が同じ
- [ ] ページの役割が区別できない
- [ ] 英語ラベルが装飾になっている
- [ ] ウェブサイト風UIをそのまま使う
- [ ] 読み手が何を判断すべきか分からない
- [ ] 発表用、閲覧用、会議用の区別がない

## 画像・図

- [ ] 画像を消しても理解が変わらない
- [ ] 抽象概念を抽象画像へ置き換える
- [ ] 発光、粒子、ワイヤーフレームに依存する
- [ ] 図解が本文の再掲になっている
- [ ] アイコンや色分けに意味がない
- [ ] 写真の日時、場所、出所が不明
- [ ] チャートの単位、母数、期間が不明
- [ ] 生成画像を事実画像と誤認しうる
- [ ] 別テーマでも使い回せる素材である

## 根拠・論理

- [ ] 事実と解釈が混ざる
- [ ] 強い主張の根拠が遠い
- [ ] 反証がない
- [ ] 限界が最後に小さくまとめられる
- [ ] 対象範囲、時点、比較条件が不明
- [ ] 因果と相関を混同する
- [ ] 参考文献が内容と結びついていない
- [ ] 出典が特定言語、特定分野へ偏る

## 判断主体

- [ ] 誰が結論を選んだか分からない
- [ ] なぜこの構成・画像なのか説明できない
- [ ] 不採用案がない
- [ ] AIの初稿をほぼそのまま使う
- [ ] 読み手固有の情報がない
- [ ] 使用場面や制約が反映されていない
- [ ] 制作者の観察が残っていない
- [ ] 最終判断者が不明

## 目安

- 0〜15：個別表現と根拠を確認
- 16〜30：具体性、構造、画像の役割を見直す
- 31〜50：原材料へ戻り、構成から再検討
- 51以上：テイスト変更ではなく、目的と結論の選定から作り直す

## 最終5問

1. この成果物でしか伝えられない情報は何か
2. 制作者が実際に判断した箇所はどこか
3. 読み手に何を判断・理解・実行してほしいか
4. どこまでが事実で、どこからが解釈か
5. なぜこの文章・配置・画像でなければならないか
