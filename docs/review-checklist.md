# AIくささレビュー・チェックリスト

各項目を0（問題なし）、1（少し該当）、2（明確に該当）で確認する。点数は検出器ではなく、改善箇所の発見に使う。

## 段階制作・早すぎる収束

- [ ] 原材料から、いきなり完成版を作っている
- [ ] 目的や読み手を確認する前に、テイストを決めている
- [ ] 構成案を比較せず、最初の案を採用している
- [ ] 構成が未確定のまま、細かな装飾を作り込んでいる
- [ ] グリッドや文字体系を試す前に、全ページを量産している
- [ ] 静、標準、密、動の代表ページを試していない
- [ ] 重要な段階でユーザーまたは制作者の判断を挟んでいない
- [ ] 完成後の自己評価だけで、途中の判断を検証していない

## 文章

- [ ] 抽象語が具体語より多い
- [ ] 「AではなくB」が複数回ある
- [ ] すべての見出しがキャッチコピー風
- [ ] 文の長さが均一
- [ ] 同じ接続詞が続く
- [ ] 結論が毎回きれいにまとまりすぎる
- [ ] 固有名詞、数字、日時が少ない
- [ ] 実際の発言や観察がない
- [ ] 誰にでも当てはまる一般論が多い
- [ ] 書き手の判断や迷いが見えない

## 構造・レイアウト

- [ ] 同じ構成が3ページ以上続く
- [ ] すべて3項目に整理されている
- [ ] 内容に関係なくカードを使う
- [ ] 全ページの情報密度が同じ
- [ ] ページの役割が区別できない
- [ ] 英語ラベルが装飾になっている
- [ ] ウェブサイト風UIをそのまま使う
- [ ] 読み手が何を判断すべきか分からない
- [ ] 発表用、閲覧用、会議用の区別がない
- [ ] グリッドが関係の設計ではなく、同じ箱の反復になっている
- [ ] 前後のページで理解がどう変わるか説明できない

## 参照・独自性

- [ ] 参照資料を「この感じで」という雰囲気だけで使っている
- [ ] 借りる操作と借りない外見を区別していない
- [ ] 骨格、逸脱、時間、固有素材の参照が一つの種類に偏っている
- [ ] 有名作品の配色、書体、質感を表面的に模倣している
- [ ] 題材固有の発言、数字、写真、表、手書きが中心になっていない
- [ ] 別テーマへそのまま使い回せる
- [ ] 一度目より二度目に意味が増える要素がない
- [ ] 面白味を、奇抜な装飾だけで作っている

## リズム・ダイナミックさ

- [ ] すべてのページが同じ速度で読まれる
- [ ] 低密度、標準密度、高密度の差がない
- [ ] すべてのページを強く、派手にしている
- [ ] 逸脱ルールが複数あり、ページごとに別の奇抜さを使う
- [ ] 逸脱と内容上の転換が結びついていない
- [ ] 拡大、切断、反復、反転などを一ページで使いすぎる
- [ ] 静かなページや余韻がない
- [ ] 前に置いた要素の反復や回収がない

## 画像・図

- [ ] 画像を消しても理解が変わらない
- [ ] 抽象概念を抽象画像へ置き換える
- [ ] 発光、粒子、ワイヤーフレームに依存する
- [ ] 図解が本文の再掲になっている
- [ ] アイコンや色分けに意味がない
- [ ] 写真の日時、場所、出所が不明
- [ ] チャートの単位、母数、期間が不明
- [ ] 生成画像を事実画像と誤認しうる
- [ ] 別テーマでも使い回せる素材である
- [ ] 紙テクスチャ、版ずれ、手書き風を人間らしさの代用品にしている

## 根拠・論理

- [ ] 事実と解釈が混ざる
- [ ] 強い主張の根拠が遠い
- [ ] 反証がない
- [ ] 限界が最後に小さくまとめられる
- [ ] 対象範囲、時点、比較条件が不明
- [ ] 因果と相関を混同する
- [ ] 参考文献が内容と結びついていない
- [ ] 出典が特定言語、特定分野へ偏る

## 判断主体

- [ ] 誰が結論を選んだか分からない
- [ ] なぜこの構成、グリッド、画像なのか説明できない
- [ ] 不採用案がない
- [ ] AIの初稿をほぼそのまま使う
- [ ] 読み手固有の情報がない
- [ ] 使用場面や制約が反映されていない
- [ ] 制作者の観察が残っていない
- [ ] 最終判断者が不明
- [ ] 各ステージで何を確定したか記録がない
- [ ] 代表ページで何を確認したか説明できない

## 点数より先に見る危険信号

以下のいずれかに該当する場合、表面的な修正ではなく前のステージへ戻る。

- 構成案を比較していない
- 代表ページを試していない
- 題材固有の原材料がない
- 参照先から借りた操作を説明できない
- 逸脱の理由を説明できない
- 読み手に求める判断が不明

## 戻るステージの目安

- 目的、読み手が曖昧：STEP 1へ戻る
- 具体性、根拠が弱い：STEP 2へ戻る
- 既視感、模倣感が強い：STEP 7へ戻る
- 話が平坦、論理が弱い：STEP 4〜5へ戻る
- きれいだが面白くない：STEP 7〜8へ戻る
- ページごとの完成度に差がある：STEP 8へ戻る
- 装飾過多、重複が多い：STEP 9〜10へ戻る

## 最終10問

1. この成果物でしか伝えられない情報は何か
2. 制作者が実際に判断した箇所はどこか
3. 読み手に何を判断、理解、実行してほしいか
4. どこまでが事実で、どこからが解釈か
5. なぜこの文章、配置、画像でなければならないか
6. 完成前に、どの段階で何を検証したか
7. 参照先から何の操作を借り、何の外見を借りなかったか
8. グリッドは何を関係づけているか
9. 逸脱は、どの内容上の転換を表しているか
10. 資料全体の静、密、動、余韻はどこにあるか
