# 共通原則

## 人間らしさは雑さではなく判断

手書き風フォント、紙の質感、曲げた線、不規則な余白、口語を機械的に加えない。それらは理由がなければ「人間らしさの装飾」にすぎない。

人間らしさは、例の選択、事実の重みづけ、留保、削除、例外、順序などの判断から生まれる。

## 原材料を消しすぎない

実際の発言、会話、日時、場所、人数、数値、固有名詞、失敗、迷い、反応、例外、予想外だった点を残す。

抽象化は必要だが、すべてを「価値」「体験」「本質」「可能性」へ変換しない。

## 事実・解釈・提案を分ける

- 事実：確認できたもの
- 解釈：事実から読み取ったもの
- 仮説：まだ検証していないもの
- 提案：今後行うべきこと
- 演出：理解や印象を補助するもの

すべてを同じ確信度、同じ見た目で扱わない。

## 一貫性と均質性を区別する

書体、色、余白、表記、出典形式は一貫させる。文字量、構図、見出しの強さ、項目数、結論の調子は内容に応じて変える。

## 媒体に合わせる

- ウェブ：読み手が自分の速度で選択する
- スライド：発表者が時間軸を制御する
- 文書：単独で読んでも理解できる
- SNS：文脈が切り取られることを前提にする

別媒体の見た目だけを移植しない。

## 反証と限界を結論の近くに置く

最後に小さく免責を書くのではなく、主張、根拠、言える範囲、言えない範囲を近接させる。

## AIの役割を限定する

AIへ任せやすいのは候補、抜け、校正、形式、比較軸、反論、体裁。人間が保持するのは目的、読み手、原材料、優先順位、結論、採否、責任。
