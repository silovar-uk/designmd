# 素材設計：質感を装飾で終わらせない

> 状態：stable  
> designmd版：2026.07.19-beta  
> 最終更新：2026-07-19

質感は、紙の粒子画像やノイズを背景へ重ねることではない。

> 素地、インク、光、加工、構造、経年状態、画面上の振る舞いを組み合わせ、成果物がどのような物として存在しているかを設計する。

編集構造が読む時間をつくり、素材設計が世界の温度をつくり、制作痕跡が誰がどう作ったかを感じさせる。

## 1. 三つの層を分ける

### 編集構造

どう読ませるか。詳細は[`editorial-design.md`](editorial-design.md)を参照する。

### 素材感

何でできているように感じるか。

- 乾いている／湿っている
- 粗い／滑らか
- 光を吸う／反射する
- 薄い／厚い
- 軽い／重い
- 白い／生成り／灰色がかっている
- 透ける／遮る

### 制作痕跡

どう作られたように見えるか。

- オフセット印刷
- 新聞印刷
- コピー機
- リソグラフ
- 活版
- インクジェット
- 手書き
- 切り貼り
- 折り、綴じ、裁断
- エンボス、デボス、箔、ニス

## 2. 質感を構成する七要素

### 1. 素地

- 白色度
- 色温度
- 繊維感
- 粗さ
- 厚み
- 透過性
- 均一性

「紙っぽい」ではなく、自然な生成りの非塗工紙、白色度の高い平滑紙、薄く透ける紙など、物性を分けて書く。

### 2. インクの乗り方

- 輪郭が鋭い
- 少しにじむ
- 黒が沈む
- 黒が浮く
- 網点が見える
- 色が重なる
- 版がわずかにずれる
- インク量にむらがある

紙の粒子だけを重ね、文字や写真のエッジが完全なデジタルのままだと、素材と印刷が分離して見える。

### 3. 光の反応

- マット
- サテン
- グロス
- 金属的
- 半透明
- 光を吸収する
- 角度で見え方が変わる

光沢は高級感の記号ではない。写真、操作要素、注目箇所など、役割の異なる層を分けるために使う。

### 4. 加工

- 箔押し
- 空押し
- エンボス
- デボス
- 型抜き
- ニス
- ラミネート
- ミシン目
- 折り
- 小口

加工は装飾量を増やすためではなく、触る場所、開く場所、注目する場所を決める。

### 5. 構造

- 中綴じ
- 無線綴じ
- 糸綴じ
- 新聞折り
- 観音開き
- 蛇腹
- 別紙挿入
- 半透明紙の重なり
- ページサイズの変化

質感は一ページの表面だけではない。めくったときに何が隠れ、重なり、現れるかを含む。

### 6. 経年状態

- 新品
- 日常的に使われている
- 読み込まれている
- 日焼けしている
- 保管されていた
- 折り目がある
- 書き込みがある
- 補修されている

「ヴィンテージ」にまとめず、誰が、いつ、どう扱った物かを決める。根拠のない汚れや傷を足さない。

### 7. 画面上の振る舞い

- ページが滑る
- 紙が重なる
- 写真が裁ち落とされる
- 半透明層越しに次の内容が見える
- スクロールで注釈が現れる
- 折り目を境に情報が切り替わる

物理物を忠実に模倣する必要はない。理解や操作を助ける物性だけを、色、影、重なり、動きへ翻訳する。

## 3. 五つの連続軸

質感は完成済みの五種類ではなく、五本の軸で指定する。

### 軸1：表面

```text
1 平滑 ─ 2 微細な粒子 ─ 3 繊維質 ─ 4 ざらつき ─ 5 強い凹凸・摩耗
```

### 軸2：反射

```text
1 光を吸う ─ 2 低反射 ─ 3 半光沢 ─ 4 光沢 ─ 5 鏡面・金属
```

### 軸3：精度

```text
1 完全に均一 ─ 2 わずかな揺れ ─ 3 にじみ・網点 ─ 4 版ずれ・潰れ ─ 5 強い手作業のむら
```

### 軸4：時間

```text
1 未使用 ─ 2 新刊 ─ 3 日常使用 ─ 4 保管・日焼け ─ 5 劣化・補修
```

### 軸5：重量

```text
1 薄く透ける ─ 2 軽い ─ 3 標準 ─ 4 厚い ─ 5 硬く重い
```

案件別には次の形で記録する。

```text
表面：4 / 5　やや粗い
反射：1 / 5　ほぼ完全なマット
精度：3 / 5　わずかなにじみと網点
時間：2 / 5　新品だが素朴
重量：2 / 5　薄く軽い
制作痕跡：新聞印刷、中央の折り目
```

## 4. 編集型との典型的な組み合わせ

固定ルールではなく、最初の仮説として使う。

### 報道・新聞型

- 低白色度
- マット
- 薄い
- 少し吸水性を感じる
- 黒がわずかににじむ
- 細かな網点
- 折り目

避ける：強い汚れ、全面の版ずれ、読めないほどの新聞再現。

### 写真・ラグジュアリー型

- 平滑
- 高密度
- 深い黒
- 写真部分だけわずかに光沢
- 本文部分はマット
- 厚めの表紙
- 正確な裁断

避ける：全面グロス、金色と影の多用、すべての要素への加工。

### 文芸・思想誌型

- 自然な生成り
- 非塗工
- 細かな繊維感
- 低反射
- 柔らかい黒
- 適度な厚み

避ける：セピア、古書風の染み、本文背後の強い紙粒子。

### カルチャー・インディー型

- コピー粒子
- 網点
- 切り貼り
- テープ
- ホチキス
- 版ずれ
- 黒の潰れ
- 一部だけの特色

避ける：全ページへのランダムな汚し、本文の常時読みにくさ。

### 解説・データ型

- 白または薄いグレー
- 比較的平滑
- シャープな線と数字
- 半透明レイヤー
- インデックス、タブ、注釈
- 重なりによる関係表現

避ける：無意味な方眼紙、設計図風の座標、装飾だけの数字。

## 5. 質感を言葉にする

次の五点を一組で書く。

1. 何の素材か
2. 表面はどうなっているか
3. 光をどう受けるか
4. どの制作痕跡が残っているか
5. どの時間状態にあるか

### 弱い指定

```text
雑誌っぽい紙の質感。
少しレトロで高級感がある。
```

### 具体的な指定

```text
自然な生成り色の非塗工紙を想定する。
表面には細かな繊維感があるが、汚れや大きな凹凸はない。
光沢はなく、黒インクはわずかに紙へ沈む。
裁断と印刷は正確で、手作り感や版ずれは加えない。
古書ではなく、丁寧に作られた新刊の文芸誌の状態とする。
```

## 6. 画面への翻訳

### 再現する

- 色温度
- コントラスト
- 表面の粗さの程度
- 重なりと奥行き
- ページ送りの速度

### 抽象化する

- 紙の厚み → 影、重なり、切り替え速度
- 透け → 次の内容の予告
- 折り → 情報の区切り
- 綴じ → ナビゲーションの基準線
- 加工 → 限定された強調

### 捨てる

- 読解を妨げるノイズ
- 根拠のない汚れ
- 操作できない装飾的な紙の端
- 画面サイズと合わない物理的縮尺
- 内容と関係しないめくり演出

## 7. 案件別スタイルガイド用テンプレート

```text
【素材設計】
想定する物体：
素地：色、白色度、粗さ、厚み、透過性
インク：輪郭、濃度、にじみ、網点、版ずれ
光：マット、半光沢、光沢、金属、透明
加工：箔、エンボス、ニス、型抜き、折り
構造：綴じ方、開き方、重なり、ページ差
経年：新品、使用済み、保管、日焼け、補修
五軸：表面 / 反射 / 精度 / 時間 / 重量
制作痕跡：一つまたは二つ
画面への翻訳：画像、色、影、動き、重なりのどれで表すか
禁止事項：使わない質感、加工、演出
```

## 8. レビュー

- [ ] 紙テクスチャを全面へ敷いただけになっていない
- [ ] 素地とインクの精度が矛盾していない
- [ ] 光源や影の方向がページごとに変わっていない
- [ ] 経年状態を説明できる
- [ ] 汚れ、傷、版ずれに内容上の理由がある
- [ ] 高級感を金色、細い書体、影だけで作っていない
- [ ] 質感を削除した場合に失われる意味を説明できる
- [ ] 画面では操作と読解を優先している
- [ ] 制作痕跡を一つまたは二つへ絞っている
